💡 Why GitHub Gist?

✅ FREE — GitHub gives unlimited private Gists at zero cost
✅ PERMANENT — Data never lost, survives Render restarts, redeploys, sleeps
✅ SECURE — Private Gist, only you can access with your GitHub token
✅ NO SETUP — Auto-created on first run, just add 2 env vars to Render
✅ EASY MIGRATION — When ready for Phase 2, export JSON to Supabase/Firebase
⚠️ LIMITS — Good up to ~10,000 users. After that, move to Supabase (₹0 free tier)
