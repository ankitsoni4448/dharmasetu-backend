💳 Razorpay / Payment Setup

For real payments, add these to Render environment variables:

RAZORPAY_KEY_ID = rzp_live_XXXXXXXXXXXXXXXX
RAZORPAY_KEY_SECRET = your secret key
ADMIN_PASSWORD = your admin password
Get keys from: dashboard.razorpay.com → Settings → API Keys. Free account, no monthly fees, 2% per transaction.
